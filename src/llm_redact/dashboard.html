Upstreams

Custom upstreams are OpenAI-compatible backends served under /custom/<name>/ (point the tool's OPENAI_BASE_URL there); names are lowercase [a-z0-9-].

Saving rewrites the config file (comments are not preserved; one .bak of the previous file is kept) and applies immediately — no restart needed for these settings.
